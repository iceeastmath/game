# 소개
학생 수학체험 게임 HTML 코드 모음입니다. AI는 제미나이나 챗지피티도 가능하지만 가능하면 클로드를 추천드립니다.

# 구분
일반코드: 별도 조작 없이 쉽게 바로 사용할 수 있는 코드입니다. (@@@)<br>
랭킹코드: 학생들이 반과 이름을 작성하면 점수를 구글시트로 전송받아 관리할 수 있는 코드입니다. (@@@_rank)<br>
로그인코드: 랭킹코드에서 로그인 기능이 포함된 코드입니다. (@@@_login)<br>
기능이 추가될수록 어려워집니다. 처음이면 일반코드 먼저 해보세요.

# 일반코드 사용법
[쉬움]
1. 구글 사이트(sites.google.com) 접속, 로그인<br>
2. '새 사이트 시작'으로 사이트 생성<br>
3. 우측 상단 '삽입(<> 모양)' 클릭, '소스코드(URL사용 옆에)' 클릭<br>
4. 코드를 복사한 뒤 붙여넣기.<br>
5. 우측 최상단 '게시' 클릭 → 웹주소 입력 → 검색엔진 노출금지 체크 → 게시 <br>
6. sites.google.com/view/방금입력한웹주소 이 주소로 컴퓨터, 폰 모두 접속할 수 있습니다.<br>
7. (선택)구글 사이트는 사이트를 꾸미는 편집 기능이 많습니다. 직접 다뤄보며 편집해보기 바랍니다.<br>

[어려움]
1. 네트리파이(netlify), 깃허브, 구글 드라이브를 이용하는 방법이 있는데 이 부분은 인공지능에게 물어보며 배워보는게 좋습니다.

# 랭킹코드 사용법
1. 코드를 복사한 뒤 AI에게 붙여넣기 하고 '이 게임 html 코드는 수정하지 말고 이 코드에 맞게 반과 이름, 점수를 구글 시트로 전송하는 구글 앱스크립트 코드를 작성해줘'라고 보냅니다.<br>
2. '구글 앱스크립트 사용법을 몰라. 처음 다루는 사람에게 설명하듯 하나하나 자세히 설명해줘.'라고 보내고 하라는대로 따라합니다.<br>
3. 대략 이런 과정일겁니다. 구글 시트 생성 → 시트ID를 앱스크립트에 붙여넣기(이 과정은 생략될 수 있음) → 앱스크립트 배포 → HTML 코드 속에 URL 붙여넣기<br>
4. 최종 수정된 HTML 코드를 구글 사이트에 붙여넣기 하여 사용하면 됩니다. 이후 과정은 #일반코드 사용법과 같습니다.<br>

# 로그인코드 사용법
1. 코드를 복사한 뒤 AI에게 붙여넣기 하고 '이 게임 html 코드는 수정하지 말고 이 코드에 맞게 학번과 이름으로 로그인하는 시스템(구글 시트와 대조하는 방식)과 점수를 구글 시트로 전송하는 구글 앱스크립트 코드를 작성해줘'라고 보냅니다.<br>
2. 마찬가지로 잘 모르니까 자세히 설명하라고 AI에게 부탁합니다.<br>
3. 구글 시트 생성 → 로그인용 시트, 점수기록 시트 생성 → 로그인용 시트에 학생 학번과 이름 전부 입력 → 시트ID, 시트이름 앱스크립트에 붙여넣기 → 앱스크립트 배포 → HTML 코드 속에 URL 붙여넣기<br>
4. 최종 수정된 HTML 코드를 구글 사이트에 붙여넣기 하여 사용하면 됩니다. 이후 과정은 #일반코드 사용법과 같습니다.<br>
