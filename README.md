# 소개
학생 수학체험 게임 html 코드 모음입니다.
파일 이름에 따라 일반코드, 랭킹코드, 로그인코드로 구분합니다.
일반코드는 파일명 뒤에 아무것도 없는 코드입니다.
랭킹코드는 파일명 뒤에 rank로 끝나는 코드입니다.
로그인코드는 파일명 뒤에 login으로 끝나는 코드입니다.
제미나이나 챗지피티도 가능하지만 되도록 클로드를 추천드립니다.

# 일반코드 사용법
[쉬움]
1. 구글 사이트(sites.google.com) 접속, 로그인
2. '새 사이트 시작'으로 사이트 생성
3. 우측 상단 '삽입(<> 모양)' 클릭, '소스코드(URL사용 옆에)' 클릭
4. 코드를 복사한 뒤 붙여넣기.
5. 적절히 편집한 뒤(구글 사이트 편집 요소가 많습니다. 직접 다뤄보시기 바랍니다.) 우측 최상단 '게시' 클릭
6. 해당하는 사이트 주소로 접속할 수 있습니다.

[어려움]
1. 네트리파이(netlify), 깃허브, 구글 드라이브를 이용하는 방법이 있는데 이 부분은 인공지능에게 물어보며 배워보시는게 좋습니다.

# 랭킹코드 사용법
랭킹코드는 학생들이 게임을 즐기고 반과 이름을 작성하면 데이터가 구글 시트로 저장되는 코드입니다.
1. 코드를 복사한 뒤 AI에게 붙여넣기 하고 '이 게임 html 코드는 수정하지 말고 이 코드에 맞게 반과 이름, 점수를 구글 시트로 전송하는 구글 앱스크립트 코드를 작성해줘'라고 보냅니다.
2. AI가 알려준대로 따라합니다. 구글 시트 생성 > 시트 정보 앱스크립트에 붙여넣기(이 과정은 생략될 수도 있습니다) > 앱스크립트 배포(여기서 AI가 따라하는걸 정확하게 따라해야 버그가 발생하지 않습니다.) > URL 코드에 붙여넣기
3. 최종 수정된 html 코드를 구글 사이트에 붙여넣기 하여 사용하면 됩니다.(이후 과정은 #일반코드 사용법과 같음)

# 로그인코드 사용법
로그인코드는 랭킹코드에서 본인의 이름을 안쓰고 장난치기 시작하며 은어와 비속어가 올라와서 만든 시스템입니다. 랭킹코드보다 조금 어렵습니다.
1. 코드를 복사한 뒤 AI에게 붙여넣기 하고 '이 게임 html 코드는 수정하지 말고 이 코드에 맞게 학번과 이름으로 로그인하는 시스템(구글 시트와 대조)과 점수를 구글 시트로 전송하는 구글 앱스크립트 코드를 작성해줘'라고 보냅니다.
2. AI가 알려준대로 따라합니다. 구글 시트 생성 > 로그인용 시트, 점수기록 시트 생성 > 시트 정보 앱스크립트에 붙여넣기 > 앱스크립트 배포(여기서 AI가 따라하는걸 정확하게 따라해야 버그가 발생하지 않습니다.) > URL 코드에 붙여넣기
3. 최종 수정된 html 코드를 구글 사이트에 붙여넣기 하여 사용하면 됩니다.(이후 과정은 #일반코드 사용법과 같음)
